NAME

Cicada - Developer first software solutions.

DESCRIPTION

Cicada is building developer first tooling, utilities, platforms and other software solutions for the modern day developer. Our goal is to provide developers with tools that are:

  • Free and Open Source
  • Secure by default
  • Unix Philosophy inspired
  • Interoperable, using existing standards whenever possible

We are seeing a real-time erosion of industry standard technologies and companies which developers rely on. Cicada is rebuilding these technologies to meet the demands of modern developers, creating better tools by learning from the mistakes of our modern tools.

Some of the technologies we are looking at rebooting include:

  • Version control systems like GitHub, GitLab and BitBucket
  • Critical infrastructure such as OCI and package registries
  • Managed cloud environments such as AWS and GCP
